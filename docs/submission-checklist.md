## üì∏ Screenshot Taking Tips

**Tools:**
- Mac: Cmd+Shift+4 (select area), Cmd+Shift+3 (full screen)
- Windows: Windows+Shift+S (Snipping Tool)
- Browser extension: "Full Page Screen Capture"

**Editing:**
- Use Preview (Mac) or Paint (Windows) to crop to 1600x1000
- Or use online tool: https://www.iloveimg.com/crop-image

**Best Practices:**
- Use actual data (not lorem ipsum)
- Show real bookings in calendar
- Make UI look clean (close unnecessary browser tabs)
- Ensure text is readable
- Show the app actually working
